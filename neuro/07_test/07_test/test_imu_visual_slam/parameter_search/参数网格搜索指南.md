## 其他重要参数（建议手动调整或创建搜索脚本）

### 4. VT匹配阈值
**当前值**: 0.06 (Baseline和Ours相同)

**建议搜索范围**: [0.04, 0.05, 0.06, 0.07, 0.08]

**影响**: 
- 阈值越小 → VT数量越多 → 更多回环检测机会
- 阈值越大 → VT数量越少 → 更少误匹配

---

### 5. 经验地图参数

- `exp_loops`: 3 (图优化迭代次数)
- `exp_correction`: 0.12 (校正步长)

**建议搜索范围**:

- `exp_loops`: [1, 2, 3, 4, 5]
- `exp_correction`: [0.08, 0.10, 0.12, 0.15, 0.18]

## 推荐搜索顺序

1. **第一步**: 运行 `GRID_SEARCH_IMU_FUSION.m`
   - 确定最优的IMU融合权重
   - 这是最重要的参数，影响最大

2. **第二步**: 运行 `GRID_SEARCH_LSTM_GATES.m`
   - 禁用IMU，专注测试VT模块
   - 确定LSTM门控最优值

3. **第三步**: 运行 `GRID_SEARCH_TRANSFORMER_WEIGHTS.m`
   - 禁用IMU，专注测试VT模块
   - 确定Transformer增强权重

4. **第四步**: 综合最优参数
   - 将步骤1-3的最优参数组合
   - 运行完整实验验证
