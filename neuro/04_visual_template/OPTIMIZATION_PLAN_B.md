# HART+Transformer 优化 Plan B

## 🔄 V2结果反思

### V2配置失败原因

**配置**：
```
全局权重: 0.15 ✅
VT阈值: 0.07 ❌ (过高)
```

**结果**：
```
VT数量: 275 (目标290-320) ❌ 不足
经验节点: 398 (目标410-450) ❌ 不足
RMSE: 152.9m (目标<145m) ❌ 未改善
```

**问题诊断**：
```
减少权重(0.3→0.15) → 特征区分度↑ ✅
     +
提高阈值(0.06→0.07) → 匹配更严格 ❌
     ↓
双重作用 → VT生成过少 → 地图稀疏 → RMSE无改善
```

### 关键教训

```
✅ 正确：减少全局调制权重(0.15)提升了特征区分度
❌ 错误：同时提高阈值过度补偿

教训：
一次只改一个变量！
- 先改权重，阈值保持
- 观察效果后再决定是否调阈值
```

**单变量原则**：
```
✅ 正确：一次只改一个参数
❌ 错误：同时改多个参数（V2的失误）

例子：
V1→V2: 同时改权重和阈值 → 失败 ❌
V1→Plan B: 只改权重 → 有望成功 ✅
```

**对比验证**：
```
改参数前：记录baseline
改参数后：对比变化
判断：单一参数的实际影响
```
